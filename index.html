Proyecto Kampom

Kampom es una APP que te permite organizar tu tiempo y tus tareas, de manera efectiva.

Desarrollado bajo las tecnicas de trabajo KANBA y Pomodoro, genera un tablero para poder generar tareas a realizar y los estados de cada una de ellas. Ademas se complementa con un Temporizador de tiempo y descanso.

Lenguajes: Html, Css, JavaScript (LocalStorage)
