Proyecto Kampom

Kampom es una APP que te permite organizar tu tiempo y tus tareas, de manera efectiva.

Desarrollado bajo las tecnicas de trabajo KANBA y Pomodoro, genera un tablero para poder generar tareas a realizar y los estados de cada una de ellas. Además se complementa con un Temporizador de tiempo y descanso.

Sitio: www.kanpom.es

Lenguajes: Html, Css, JavaScript (LocalStorage)

Proyecto Rompecabezas

Rompecabezas es una APP educativa para niños que permite armar rompecabezas de energías renovables.

Desarrollado en JavaScript, permite a los niños aprender mientras juegan, fomentando el uso de energías renovables a través de una actividad lúdica.

Sitio: www.rompecabezas.es

Lenguajes: Html, Css, JavaScript (LocalStorage)
